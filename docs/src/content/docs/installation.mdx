## Quick Install

Install Refaktor on Linux or macOS:

```bash
curl -fsSL https://docspring.github.io/refaktor/install.sh | bash
```

### Alternative Install Options

```bash
# System-wide install
curl -fsSL https://docspring.github.io/refaktor/install.sh | bash -s -- --system

# Custom directory
curl -fsSL https://docspring.github.io/refaktor/install.sh | bash -s -- --prefix "$HOME/bin"

# Uninstall
curl -fsSL https://docspring.github.io/refaktor/install.sh | bash -s -- --uninstall
```

The install script defaults to `~/.local/bin` (no sudo required) and detects your OS and architecture automatically.

## Build from Source

For developers who want to build from source:

**Prerequisites**: Rust 1.89.0 or later

```bash
git clone https://github.com/DocSpring/refaktor
cd refaktor
cargo build --release
cp ./target/release/refaktor /usr/local/bin/
```
