## Quick Install

Install Refaktor on Linux or macOS:

```bash
# User-local install (default, no sudo needed)
curl -fsSL https://docspring.github.io/refaktor/install.sh | bash

# System-wide install
curl -fsSL https://docspring.github.io/refaktor/install.sh | bash -s -- --system

# Custom directory
curl -fsSL https://docspring.github.io/refaktor/install.sh | bash -s -- --prefix "$HOME/bin"

# Uninstall
curl -fsSL https://docspring.github.io/refaktor/install.sh | bash -s -- --uninstall
```

The install script:
- Defaults to `~/.local/bin` (no sudo required)
- Detects your OS and architecture automatically
- Creates the directory if it doesn't exist
- Provides shell-specific PATH instructions if needed
- On macOS with Homebrew, suggests `brew install refaktor` (coming soon)

## Manual Installation

### Linux

#### x86_64 / amd64
```bash
curl -L https://github.com/DocSpring/refaktor/releases/latest/download/refaktor-linux-amd64.tar.gz | tar xz
sudo mv refaktor /usr/local/bin/refaktor
sudo chmod 755 /usr/local/bin/refaktor
```

#### arm64
```bash
curl -L https://github.com/DocSpring/refaktor/releases/latest/download/refaktor-linux-arm64.tar.gz | tar xz
sudo mv refaktor /usr/local/bin/refaktor
sudo chmod 755 /usr/local/bin/refaktor
```

## Build from Source

For developers who want to build from source:

**Prerequisites**: Rust 1.89.0 or later

```bash
git clone https://github.com/DocSpring/refaktor
cd refaktor
cargo build --release
cp ./target/release/refaktor /usr/local/bin/
```

## Verify Installation

```bash
refaktor --help
```

You should see the help output showing available commands and options.
