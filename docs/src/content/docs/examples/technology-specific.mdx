# Technology-Specific Examples

## React/TypeScript Projects

### Component Refactoring
```bash
# Rename a React component across the entire project
refaktor rename UserCard ProfileCard \
  --include "src/**/*.{tsx,ts,js,jsx}" \
  --exclude "**/*.test.*"

# Or use specific paths
refaktor rename UserCard ProfileCard src/components/ src/pages/
```

**Changes made:**
- Component files: `UserCard.tsx` → `ProfileCard.tsx`
- Test files: `UserCard.test.tsx` → `ProfileCard.test.tsx` (if not excluded)
- Import statements: `import UserCard from './UserCard'`
- JSX usage: `<UserCard />` → `<ProfileCard />`
- Type definitions: `UserCardProps` → `ProfileCardProps`

### Hook Refactoring
```bash
# Rename a custom React hook
refaktor rename useUserData useProfileData \
  --include "src/**/*.{ts,tsx}"

# Or target specific directories
refaktor rename useUserData useProfileData src/hooks/ src/components/
```

**Changes made:**
```tsx
// Before
function useUserData(id: string) { ... }
const { data, loading } = useUserData(userId);

// After  
function useProfileData(id: string) { ... }
const { data, loading } = useProfileData(userId);
```

## Python Projects

### Django Model Refactoring
```bash
# Rename Django models and related code
refaktor rename UserProfile AccountProfile \
  --include "**/*.py" \
  --exclude "migrations/**"

# Or target specific app directories
refaktor rename UserProfile AccountProfile apps/users/ apps/accounts/
```

**Changes made:**
```python
# models.py - Before
class UserProfile(models.Model):
    def get_user_profile_display(self): ...

# models.py - After
class AccountProfile(models.Model):
    def get_account_profile_display(self): ...

# views.py - Before
def user_profile_view(request): ...
profile = UserProfile.objects.get(id=1)

# views.py - After
def account_profile_view(request): ...
profile = AccountProfile.objects.get(id=1)
```

### Flask API Refactoring
```bash
# Rename Flask route handlers
refaktor rename getUserData fetchUserProfile \
  --include "**/*.py" \
  --exclude "venv/**"

# Or target specific modules
refaktor rename getUserData fetchUserProfile app/ routes/
```

## Rust Projects

### Module Refactoring
```bash
# Rename Rust modules and functions
refaktor rename user_manager profile_manager \
  --include "**/*.rs" \
  --exclude "target/**"

# Or target specific source directories
refaktor rename user_manager profile_manager src/ lib/
```

**Changes made:**
```rust
// Before
mod user_manager {
    pub fn get_user_manager() -> UserManager { ... }
}
use user_manager::UserManager;

// After
mod profile_manager {
    pub fn get_profile_manager() -> ProfileManager { ... }
}
use profile_manager::ProfileManager;
```

### Cargo.toml Updates
```bash
# Rename crate references
refaktor rename my_crate awesome_crate \
  --include "**/Cargo.toml,**/*.rs"
```

## Database Projects

### SQL Schema Refactoring
```bash
# Rename database tables and columns (be careful!)
refaktor rename user_data profile_data \
  --include "**/*.sql" \
  --exclude "migrations/**" \
  --dry-run  # Always dry-run first for SQL!
```

**Changes made:**
```sql
-- Before
CREATE TABLE user_data (
    id SERIAL PRIMARY KEY,
    user_data_json JSONB
);

-- After
CREATE TABLE profile_data (
    id SERIAL PRIMARY KEY, 
    profile_data_json JSONB
);
```

## Best Practices by Technology

### Always Exclude Dependencies
- **Node.js**: `--exclude "node_modules/**"`
- **Rust**: `--exclude "target/**"`
- **Python**: `--exclude "venv/**,__pycache__/**"`
- **Go**: `--exclude "vendor/**"`

### Include Relevant Extensions
- **Web**: `--include "**/*.{js,ts,tsx,jsx,html,css,scss}"`
- **Backend**: `--include "**/*.{py,js,go,rs,rb,php}"`
- **Config**: `--include "**/*.{json,yml,yaml,toml,env}"`

### Test Safely First
```bash
# Always start with dry-run for unfamiliar codebases
refaktor dry-run pattern replacement --include "src/**"
```
